# Flutter ToDo App

A simple Flutter mobile app for registering, logging in, and viewing tasks.  

![Sign In Screen](assets/screenshots/sign_in_screen.png)
![Sign Up Screen](assets/screenshots/sign_up_screen.png)
![Home Screen](assets/screenshots/home_screen.png)

## Features
- Sign in & sign up
- View example tasks
- Logout functionality

## Built With
- Flutter & Dart
- Shared Preferences for local storage

## Running the App
```bash
git clone https://github.com/ajamajai/mobile-app.git
cd mobile-app
flutter pub get
flutter run
